# School-Bus-Tracking-and-Safety-System

School Bus Safety & Tracking System is a web-based application designed to improve the safety, monitoring, and management of school transportation. The system allows schools and parents to track buses, manage students, drivers, and routes, and receive important updates in real time.

This project is built using Python (Flask), MySQL, HTML, CSS, and JavaScript and provides separate dashboards for organizations (schools) and parents.

Key Features

Organization Dashboard

Manage students, drivers, buses, and routes

Upload and manage driver photos

Import and export student data using Excel/PDF

Monitor bus and driver information

Parent Dashboard

Secure login with email verification

View student and bus details

Track transportation information

Password reset via email

Authentication System

Secure password hashing

OTP/email verification for registration

Forgot password functionality

Data Management

MySQL database integration

Class-wise student management

Duplicate roll number validation

Technology Stack

Backend: Python (Flask)

Database: MySQL

Frontend: HTML, CSS, JavaScript

Libraries: OpenPyXL, ReportLab, Werkzeug

Purpose

The goal of this project is to provide a secure, reliable, and efficient platform for school transportation management, ensuring better safety and communication between schools and parents.
